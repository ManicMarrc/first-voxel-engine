# First Voxel Engine

My first voxel engine. Relatively smooth, gets around 60 FPS on my computer.

## Quick Start

```bash
cargo run --release
```

### Controls

| Control | Action        |
| :-----: | ------------- |
|    W    | Move forward  |
|    A    | Strafe left   |
|    S    | Move backward |
|    D    | Strafe right  |
|  Mouse  | Rotate camera |
